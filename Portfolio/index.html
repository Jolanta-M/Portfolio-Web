I’m a Computer Science graduate from Brunel University London with the passion for art and psychology, aspiring to work in UX / UI design.

During the time in university I have gained experience working in Agile teams for several projects involving web development and UX / UI design. I have developed strong skills in designing with Figma and in front-end programming using HTML5 / CSS / JavaScript and REACT / REACT NATIVE.

Project 1: Final University Project
Investigating solutions for enhancement of usefulness in personal decision support software

This project is using the research from psychology about the rational and intuitive thinking styles and their role in human decision making. Based on this research I attempted to create additional functionality for the mobile decision application so that users could get more value out of it. The new software solution was evaluated using within-subjects experimental design and UMUX questionnaire.

Project 2: From Consumer Psychology to Checkout Design: A Research-Driven Approach to Cart Abandonment

In this project for a fictional online Tea Shop, I focused on designing a shopping cart experience that draws from consumer psychology research and established UX design guidelines to reduce cart abandonment and minimise user frustration.
