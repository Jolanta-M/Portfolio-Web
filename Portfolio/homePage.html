During the time in university I have gained experience working in Agile teams for several projects involving web development and UX / UI design. I am mostly fluent in frontend programming with HTML5 / CSS / JavaScript and REACT / REACT NATIVE.

Project 1: Final University Project
Investigating solutions for enhancement of usefulness in personal decision support software

This project is using the research from psychology about the rational and intuitive thinking styles and their role in human decision making. Based on this research it was attempted to create additional functionality for the mobile decision applications so that users could get more value out of them. The new software solution was evaluated using within-subjects experimental design and UMUX questionnaire.
